pub fn extract(content: &str) -> Result<(Frontmatter, &str)>
Expand description

Extracts and parses frontmatter from content with format auto-detection.

This function provides a zero-copy extraction of frontmatter, automatically detecting the format (YAML, TOML, or JSON) and parsing it into a structured representation.

ยงPerformance

This function performs a single pass over the input with O(n) complexity and avoids unnecessary allocations where possible.

ยงExamples

use frontmatter_gen::extract;

let content = r#"---
title: My Post
date: 2025-09-09
---
Content here"#;

let (frontmatter, content) = extract(content)?;
assert_eq!(frontmatter.get("title").unwrap().as_str().unwrap(), "My Post");
assert_eq!(content.trim(), "Content here");
