pub fn extract_json_frontmatter(content: &str) -> Result<&str, FrontmatterError>
Expand description

Extracts JSON frontmatter from the content by detecting balanced curly braces ({}).

This function attempts to locate a valid JSON object starting with { and checks for balanced curly braces to identify the end of the frontmatter. If the JSON object is found, it returns the frontmatter as a string slice. A maximum nesting depth is enforced to prevent deeply nested JSON from causing stack overflow.
